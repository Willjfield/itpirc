#ITP on IRC

If you are already set up with IRC, you can connect using below commands:
/server irc.freenode.net
/join #itp

Welcome to a quick setup guide for IRC (Internet Relay Chat).
IRC is a completely free and open source platform for chattting.

The #ITP channel (chat room) is a gathering place for all ITP students and alumni to discuss and ask questions about whatever we want.

Once you set up IRC, you'll also have access to chatrooms like #processing, #arduino, #node.js, and literally thousands more! If help is needed please find AJ Levine, Will Field, or Denny George for assistance.
